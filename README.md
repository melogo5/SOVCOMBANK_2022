# SOVCOMBANK_2022

Приложение-обменник для туристов

> `nodejs`, `postgres`, `fastify`, `react`, `effector`, `javascript`, `typescript`

### настройки
Создать пустую БД в postgres (порт 5432)
...

```shell
# запуск бекенда (порт 8080)
$ cd backend/
$ npm install
$ npm start

# запуск фронтенда (порт 3000)
$ cd frontend/
$ npm install
$ npm start
```
